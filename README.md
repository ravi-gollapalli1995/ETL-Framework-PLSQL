# ETL-Framework-PLSQL

This repository contains a complete ETL (Extract, Transform, Load) implementation using **Oracle PL/SQL** and **PostgreSQL**.

## 📌 Overview

* Designed an ETL flow to load data from source tables into dimension and fact tables
* Implemented using **PL/SQL packages in Oracle**
* Converted and tested similar logic in **PostgreSQL**

## 🗂️ Project Contents

* ETL flow scripts (Source → Target)
* Oracle package (`load_tables_poc_pkg`)
* Table creation scripts
* Sample data
* PostgreSQL implementation
* Screenshots of database outputs

## ⚙️ Features

* Full load and incremental load logic
* Bulk processing (`BULK COLLECT`, `FORALL`)
* Surrogate key generation using sequences
* Basic logging using procedures
* Data transformation and joins

## 🛠️ Technologies Used

* Oracle PL/SQL
* PostgreSQL
* SQL
